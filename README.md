# react-auth-starter

react-auth-starter is a template for React projects that require user registration, login, private routes, and logout functionality

## Installation

Download the repo

```bash
cd react-client
npm install
npm start
```

```bash
cd server
npm install
npm run dev
```

## Usage

```
Open a browser and go to localhost:3000
```
